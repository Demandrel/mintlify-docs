## Prerequisites

Before you can get started, you will need to have the following installed on your machine.

* [Node.js](https://nodejs.org/en/) (v20 or higher)

* [Git](https://git-scm.com/)

* bun

* [VSCode](https://code.visualstudio.com/) (recommended, or any other code editor)

* [Android](https://tauri.app/start/prerequisites/#android) or/and [iOS](https://tauri.app/start/prerequisites/#ios) tools

## Get started

Clone the repo and install dependencies:

```
git clone https://github.com/Demandrel/supatauri.git
cd supatauri
bun install
```
